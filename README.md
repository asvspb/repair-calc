# Repair Calculator - Калькулятор ремонта

Приложение для расчета стоимости ремонтных работ.

## Для AI-агентов

**ВАЖНО:** Перед началом работы прочитайте [`INDEX.md`](./INDEX.md) — это главный индексный файл приложения.

**Правило:** После ЛЮБЫХ изменений в коде обновляйте `INDEX.md`, чтобы он всегда отражал актуальное состояние:
- Новые файлы/директории
- Изменения в типах данных
- Новые зависимости
- Изменения в архитектуре или логике
- Новые компоненты или функции

---

## Особенности

- **Многопользовательская архитектура** — проекты, объекты, комнаты
- **JWT Аутентификация** — регистрация, логин, refresh tokens
- **Автосохранение** — данные сохраняются автоматически в localStorage с синхронизацией через API
- **Экспорт/Импорт** — бэкап и восстановление через JSON
- **Excel-экспорт** — выгрузка смет в CSV для Excel
- **AI поиск цен** — Gemini и Mistral API для оценки стоимости материалов и работ
- **3 режима геометрии** — Simple, Extended, Advanced
- **Адаптивный дизайн** — работает на мобильных и десктопе

## Запуск локально

**Предварительные требования:** Node.js 18+

1. Установить зависимости:
   ```bash
   npm install
   ```

2. Создать файл `.env.local` с API ключом:
   ```
   VITE_GEMINI_API_KEY=your_gemini_api_key
   ```

3. Запустить приложение:
   ```bash
   npm run dev
   ```

4. Открыть http://localhost:3993

## Доступные команды

| Команда | Описание |

## Управление данными

### Автосохранение
- Данные автоматически сохраняются в localStorage браузера
- Сохранение происходит через 1 секунду после последнего изменения
- При авторизации — синхронизация с сервером через API
- При закрытии страницы данные также сохраняются

### Бэкап и восстановление

#### Через UI приложения
1. Нажмите кнопку **"Данные"** в правом верхнем углу
2. Выберите **"Сохранить бэкап (JSON)"** для экспорта
3. Выберите **"Загрузить бэкап (JSON)"** для импорта

#### Экспорт в Excel
- В разделе "Данные" выберите **"Экспорт в Excel (CSV)"**
- Файл CSV можно открыть в Excel или Google Sheets

### Перенос на другой компьютер

1. **Экспортируйте данные** через UI приложения (JSON)
2. **Скопируйте файл** на новый компьютер
3. **Импортируйте данные** через UI на новом компьютере

### Очистка данных
- В разделе "Данные" -> "Опасная зона" -> "Очистить все данные"
- Или через DevTools браузера: `localStorage.clear()`
