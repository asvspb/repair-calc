## Особенности

- 💾 **Автосохранение** - данные сохраняются автоматически в localStorage
- 📤 **Экспорт/Импорт** - бэкап и восстановление через JSON
- 📊 **Excel-экспорт** - выгрузка смет в CSV для Excel
- 📱 **Адаптивный дизайн** - работает на мобильных и десктопе

## Запуск локально

**Предварительные требования:** Node.js 18+

1. Установить зависимости:
   ```bash
   npm install
   ```

2. Создать файл `.env.local` с API ключом:
   ```
   GEMINI_API_KEY=your_gemini_api_key
   ```

3. Запустить приложение:
   ```bash
   npm run dev
   ```

4. Открыть http://localhost:3993

## 🕷️ Парсеры каталогов

Проект содержит парсеры для сбора данных о товарах и ценах со строительных магазинов.

### Доступные парсеры

| Парсер | Файл | Источник | Особенности |
|--------|------|----------|-------------|
| Bazavit | `bazavit-parser.js` | bazavit.ru | Битрикс, простая структура |
| Lemana PRO | `lemana-parser.js` | lemanapro.ru | SPA, защита Qrator, data-атрибуты |

### Запуск парсеров

```bash
# Установить Playwright (если ещё не установлен)
npm install playwright

# Запуск парсера Bazavit
node bazavit-parser.js

# Запуск парсера Lemana PRO
node lemana-parser.js
```

### Результаты работы

Парсеры создают файлы:
- `*_output.sql` — SQL-запросы для MySQL
- `*_data.json` — данные в формате JSON

### Особенности парсера Lemana PRO

- **Защита от ботов**: Использует Playwright с подменой User-Agent
- **SPA-архитектура**: Ждёт загрузки динамического контента
- **Формат цен**: Regex для "5 704, 60 ₽" → `5704.60`
- **Пагинация**: Автоматический переход по страницам

### ⚠️ Внимание

- Парсеры настроены на демо-режим (первые 3 категории, 3 страницы)
- Для полного парсинга измените `categories.slice(0, 3)` на `categories`
- Соблюдайте `robots.txt` и условия использования сайтов

## Управление данными

### Автосохранение
- Данные автоматически сохраняются в localStorage браузера
- Сохранение происходит через 1 секунду после последнего изменения
- При закрытии страницы данные также сохраняются

### Бэкап и восстановление

#### Через UI приложения
1. Нажмите кнопку **"Данные"** в правом верхнем углу
2. Выберите **"Сохранить бэкап (JSON)"** для экспорта
3. Выберите **"Загрузить бэкап (JSON)"** для импорта

#### Экспорт в Excel
- В разделе "Данные" выберите **"Экспорт в Excel (CSV)"**
- Файл CSV можно открыть в Excel или Google Sheets

### Перенос на другой компьютер

1. **Экспортируйте данные** через UI приложения (JSON)
2. **Скопируйте файл** на новый компьютер
3. **Импортируйте данные** через UI на новом компьютере

### Очистка данных
- В разделе "Данные" → "Опасная зона" → "Очистить все данные"
- Или через DevTools браузера: `localStorage.clear()`
