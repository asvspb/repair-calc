## Как работает v2

**При сохранении:**
```typescript
workVolume = metrics.floorArea (или netWallArea, skirtingLength)
sourceVolume = workVolume  // сохраняется в шаблон
```

**При загрузке:**
```typescript
ratio = targetVolume / sourceVolume
newQuantity = material.quantity × ratio
```

**Пример:**
| Комната | Площадь | Материал (кол-во) |
|---|---|---|
| Комната 1 (сохранение) | 10.44 м² | 45 мешков |
| Комната 2 (загрузка) | 17.64 м² | 45 × 1.69 = **76 мешков** |

## Краткое описание

При ремонте нескольких помещений многие работы повторяются (укладка ламината, шпаклевание стен, монтаж плинтусов). Нужно сохранять такие задачи как шаблоны и загружать их в новых комнатах с автоматическим пересчётом объёма.

---

## Пользовательские сценарии

### 1. Сохранение шаблона
- В карточке работы (нижний левый угол) появляется иконка **💾 Сохранить как шаблон**
- Клик → работа сохраняется под тем же названием
- Появляется уведомление: *«Шаблон сохранён»*

### 2. Загрузка шаблона
- В секции «Работы и материалы» кнопка **📋 Из шаблона**
- Открывается модальное окно со списком шаблонов
- Выбор шаблона → **Загрузить**
- Работа добавляется в комнату с теми же ценами
- **Объём автоматически пересчитывается** по метрикам нового помещения

### 3. Обновление шаблона
- Клик на 💾 при существующем шаблоне → подтверждение замены

### 4. Удаление шаблона
- В модальном окне у каждого шаблона иконка 🗑️

## Что сохраняется в шаблоне

| Поле | Сохраняется |
|---|---|
| Название работы | ✅ |
| Единица измерения | ✅ |
| Цена работы за единицу | ✅ |
| Тип расчёта объёма | ✅ |
| Материалы (название, количество, цена) | ✅ |
| Инструменты (название, количество, цена, аренда) | ✅ |

## Что НЕ сохраняется (привязано к комнате)

| Поле | Не сохраняется |
|---|---|
| `id` | ❌ Генерируется новый |
| `manualQty` | ❌ Авто-расчёт по комнате |
| `enabled` | ❌ Всегда `true` |

---

## Автоматический пересчёт объёма

При загрузке шаблона объём определяется по `calculationType`:

| Тип расчёта | Формула |
|---|---|
| `floorArea` | Площадь пола комнаты |
| `netWallArea` | Площадь стен минус проёмы |
| `skirtingLength` | Периметр минус ширина дверей |
| `customCount` | Фиксированное количество из шаблона |

**Пример:**
- Комната 1: площадь 10.44 м² → работа 10.44 × 350 = 3 654 ₽
- Комната 2: площадь 17.64 м² → работа 17.64 × 350 = 6 174 ₽

- [ ] Модальное окно поддерживает поиск и фильтрацию по категории

- [ ] UI соответствует текущему дизайну (Tailwind CSS, lucide-react, indigo-палитра)
