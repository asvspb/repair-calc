## 1. Общее описание

### 1.1 Текущее состояние
В настоящий момент система материалов работает по упрощенной схеме:
- **Тип цены**: "За весь объем" или "За единицу"
- **Цена материалов**: единое число (общая сумма или цена за единицу)

### 1.2 Проблемы текущей реализации
- Нет детализации по конкретным материалам
- Невозможно указать количество каждого материала
- Нет учета инструментов, необходимых для работы
- Сложно анализировать затраты по категориям

### 1.3 Цель доработки
Усложнить систему учета материалов и добавить учет инструментов для более точного планирования бюджета ремонта.

## 2. Требования к функционалу

### 2.1 Детализация материалов

#### 2.1.1 Структура материала
Каждый материал должен содержать следующие поля:

| Поле | Тип | Описание | Обязательное |
|------|-----|----------|--------------|
| `id` | string | Уникальный идентификатор | Да |
| `name` | string | Наименование материала | Да |
| `quantity` | number | Количество | Да |
| `unit` | string | Единица измерения (шт, кг, м², м, л и т.д.) | Да |
| `pricePerUnit` | number | Цена за единицу | Да |
| `totalPrice` | number | Итоговая стоимость (quantity × pricePerUnit) | Автоматический расчет |

#### 2.1.2 Функциональные требования
- [ ] Добавление материала к работе через кнопку "+ Добавить материал"
- [ ] Удаление материала с подтверждением
- [ ] Редактирование всех полей материала inline
- [ ] Автоматический пересчет totalPrice при изменении quantity или pricePerUnit
- [ ] Отображение суммарной стоимости всех материалов по работе
- [ ] Валидация: quantity > 0, pricePerUnit ≥ 0

#### 2.1.3 UI/UX требования
- Материалы отображаются внутри карточки работы в свернутом/развернутом виде
- Поля ввода: name (text), quantity (number), unit (select/text), pricePerUnit (number)
- Итоговая сумма по материалам выделена визуально
- Drag & drop для сортировки материалов (опционально)

### 2.2 Учет инструментов

#### 2.2.1 Структура инструмента
Каждый инструмент должен содержать следующие поля:

#### 3.2.2 Изменения в существующих компонентах

### 3.3 Функции расчета

```typescript
// Расчет стоимости материалов
const calculateMaterialsCost = (materials: Material[]): number => {
  return materials.reduce((sum, m) => sum + (m.quantity * m.pricePerUnit), 0);
};

// Расчет стоимости инструментов
const calculateToolsCost = (tools: Tool[]): number => {
  return tools.reduce((sum, t) => {
    if (t.isRent && t.rentPeriod) {
      return sum + (t.price * t.quantity * t.rentPeriod);
    }
    return sum + (t.price * t.quantity);
  }, 0);
};

// Общий расчет по работе
const calculateWorkTotal = (work: WorkData, qty: number): number => {
  const workCost = qty * work.workUnitPrice;
  const materialsCost = calculateMaterialsCost(work.materials || []);
  const toolsCost = calculateToolsCost(work.tools || []);
  return workCost + materialsCost + toolsCost;
};
```

### 7.4 Обработка ошибок при сохранении

```typescript
interface SaveResult {
  success: boolean;
  errors: ValidationError[];
  warnings: ValidationWarning[];
}

interface ValidationError {
  field: string;
  message: string;
  severity: 'error' | 'warning';
}

// Пример обработки
const handleSaveWork = (work: WorkData): SaveResult => {
  const errors: ValidationError[] = [];
  const warnings: ValidationWarning[] = [];

  // Валидация материалов
  work.materials.forEach((material, index) => {
    const validation = validateMaterial(material);
    if (!validation.valid) {
      errors.push({
        field: `materials[${index}].${validation.field}`,
        message: validation.message,
        severity: 'error'
      });
    }
  });

  // Проверка на дубликаты названий
  const names = work.materials.map(m => m.name.toLowerCase());
  const duplicates = names.filter((n, i) => names.indexOf(n) !== i);
  if (duplicates.length > 0) {
    warnings.push({
      field: 'materials',
      message: `Обнаружены дубликаты: ${duplicates.join(', ')}`
    });
  }

  return {
    success: errors.length === 0,
    errors,
    warnings
  };
};
```

## 10. Риски и ограничения

### 10.1 Технические риски

| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| Потеря данных при миграции | Низкая | Высокое | Автоматическое резервное копирование перед миграцией |
| Превышение лимита localStorage | Средняя | Среднее | Сжатие данных, предупреждение о лимите |
| Несовместимость браузеров | Низкая | Среднее | Тестирование в целевых браузерах |
| Производительность при большом количестве материалов | Низкая | Низкое | Виртуализация списка при > 50 элементов |

### 10.2 UX-риски

| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| Пользователи не понимают новую структуру | Средняя | Среднее | Туториал, подсказки, сохранение старого UI как опции |
| Сложность заполнения множества полей | Средняя | Среднее | Шаблоны материалов, автоподстановка |
| Ошибки при вводе дробных количеств | Низкая | Низкое | Подсказки формата, валидация в реальном времени |

### 10.3 Ограничения текущей реализации

1. **Нет синхронизации между устройствами** - данные хранятся локально в localStorage
2. **Нет коллективной работы** - один пользователь
3. **Нет интеграции с поставщиками** - цены вводятся вручную
4. **Нет истории изменений** - нельзя откатить изменения
5. **Нет категоризации** - материалы и инструменты не группируются

### 10.4 Ограничения по времени и ресурсам

| Ограничение | Значение |
|-------------|----------|
| Общий бюджет времени | 10-14 часов |

### Приложение Г: Примеры расчетов

#### Г.1 Расчет материалов для шпаклевания стен

**Исходные данные:**
- Площадь стен: 50 м²
- Работы: грунтование + шпаклевание финишное

**Расчет:**

```
Грунтовка:
  Площадь × Расход = 50 × 0.12 л/м² = 6 л
  Количество: 6 л → 2 банки по 5 л (с запасом)
  Цена: 2 × 450 ₽ = 900 ₽

Шпаклевка финишная:
  Площадь × Расход = 50 × 1 кг/м² = 50 кг
  Количество: 50 кг → 2 мешка по 25 кг
  Цена: 2 × 600 ₽ = 1 200 ₽

Итого материалы: 900 + 1 200 = 2 100 ₽
```

#### Г.2 Расчет аренды инструмента

**Исходные данные:**
- Работа: укладка плитки
- Площадь: 15 м²
- Срок работ: 3 дня

**Расчет:**

```
Плиткорез (аренда):
  Цена аренды: 500 ₽/день
  Срок: 3 дня
  Итого: 500 × 3 = 1 500 ₽

Миксер строительный (аренда):
  Цена аренды: 300 ₽/день
  Срок: 3 дня
  Итого: 300 × 3 = 900 ₽

Уровень (покупка):
  Цена: 800 ₽
  Количество: 1 шт
  Итого: 800 ₽

Итого инструменты: 1 500 + 900 + 800 = 3 200 ₽
```

#### Г.3 Полный расчет работы

**Исходные данные:**
- Работа: укладка ламината
- Площадь: 25 м²
- Цена работы: 350 ₽/м²

**Расчет:**

```
Стоимость работы:
  25 м² × 350 ₽/м² = 8 750 ₽

Материалы:
  Ламинат: 25 × 1.05 × 650 ₽ = 17 062.50 ₽
  Подложка: 25 × 80 ₽ = 2 000 ₽
  Плинтус: 20 м × 250 ₽ = 5 000 ₽
  Итого материалы: 24 062.50 ₽

Инструменты:
  Подбивочный брусок: 1 × 350 ₽ = 350 ₽
  Распорные клинья: 1 компл × 200 ₽ = 200 ₽
  Итого инструменты: 550 ₽

ИТОГО ПО РАБОТЕ:
  8 750 + 24 062.50 + 550 = 33 362.50 ₽
```
