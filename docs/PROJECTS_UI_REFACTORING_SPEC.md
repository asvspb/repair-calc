## 1. Обоснование изменений

### Текущие проблемы

1. **Дублирование функционала:** Список проектов дублируется в модальном окне `ProjectsModal` и частично в правой панели через `ProjectSettings`.
2. **Лишние клики:** Для переключения между проектами нужно открыть модальное окно.
3. **Непоследовательный порядок кнопок:** Кнопка "Добавить объект ремонта" находится выше кнопки "Добавить комнату", что противоречит логике иерархии (Проект → Объект → Комната).

### Цели

1. Улучшить UX: быстрый доступ к списку проектов без модального окна.
2. Разделить функционал: модальное окно — только для работы с данными (экспорт/импорт, синхронизация).
3. Исправить порядок кнопок в левой панели.
