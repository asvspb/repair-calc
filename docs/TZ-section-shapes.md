## 3. Функциональные требования

### 3.1 Поддерживаемые формы секций

#### 3.1.1 Прямоугольник (по умолчанию)
Базовая форма, уже реализована.

**Параметры:**
- Длина (м)
- Ширина (м)

**Формулы:**
- Площадь: `S = длина × ширина`
- Периметр: `P = 2 × (длина + ширина)`

#### 3.1.2 Трапеция
Для помещений со скошенными стенами, эркерами сложной формы.

**Параметры:**
- Основание 1 (м) — длинная сторона
- Основание 2 (м) — короткая сторона (параллельная)
- Высота (м) — расстояние между основаниями
- Боковая сторона 1 (м) — для расчёта периметра
- Боковая сторона 2 (м) — для расчёта периметра

**Формулы:**
- Площадь: `S = (осн1 + осн2) / 2 × высота`
- Периметр: `P = осн1 + осн2 + бок1 + бок2`

#### 3.1.3 Треугольник
Для треугольных помещений, ниш, эркеров.

**Параметры:**
- Сторона A (м)
- Сторона B (м)
- Сторона C (м)
- Высота (м) — опционально, для проверки

**Формулы:**
- Площадь: по формуле Герона `S = √(p(p-a)(p-b)(p-c))`, где `p = (a+b+c)/2`
- Периметр: `P = a + b + c`

Альтернативный ввод (упрощённый):
- Основание (м)
- Высота (м)
- Площадь: `S = основание × высота / 2`
- Периметр: вводится вручную или рассчитывается при наличии всех сторон

#### 3.1.4 Параллелограмм
Для помещений с наклонными стенами (мансарды, скошенные углы).

**Параметры:**
- Основание (м)
- Высота (м) — высота, опущенная на основание
- Боковая сторона (м)

**Формулы:**
- Площадь: `S = основание × высота`
- Периметр: `P = 2 × (основание + боковая_сторона)`

## 5. Алгоритмы расчёта

### 5.1 Функция расчёта площади секции

```typescript
function calculateSectionArea(section: RoomSubSection): number {
  switch (section.shape) {
    case 'rectangle':
      return (section.length || 0) * (section.width || 0);
    
    case 'trapezoid':
      const b1 = section.base1 || section.dimensions?.base || 0;
      const b2 = section.base2 || section.dimensions?.base2 || 0;
      const h = section.height || section.dimensions?.height || 0;
      return ((b1 + b2) / 2) * h;
    
    case 'triangle':
      // По формуле Герона
      const a = section.sideA || section.dimensions?.side1 || 0;
      const b = section.sideB || section.dimensions?.side2 || 0;
      const c = section.sideC || section.dimensions?.side3 || 0;
      const p = (a + b + c) / 2;
      return Math.sqrt(p * (p - a) * (p - b) * (p - c));
    
    case 'parallelogram':
      const base = section.base || section.dimensions?.base || 0;
      const heightVal = section.height || section.dimensions?.height || 0;
      return base * heightVal;
    
    default:
      return 0;
  }
}
```

### 5.2 Функция расчёта периметра секции

```typescript
function calculateSectionPerimeter(section: RoomSubSection): number {
  switch (section.shape) {
    case 'rectangle':
      return 2 * ((section.length || 0) + (section.width || 0));
    
    case 'trapezoid':
      const b1 = section.base1 || section.dimensions?.base || 0;
      const b2 = section.base2 || section.dimensions?.base2 || 0;
      const s1 = section.side1 || section.dimensions?.side1 || 0;
      const s2 = section.side2 || section.dimensions?.side2 || 0;
      return b1 + b2 + s1 + s2;
    
    case 'triangle':
      const a = section.sideA || section.dimensions?.side1 || 0;
      const b = section.sideB || section.dimensions?.side2 || 0;
      const c = section.sideC || section.dimensions?.side3 || 0;
      return a + b + c;
    
    case 'parallelogram':
      const base = section.base || section.dimensions?.base || 0;
      const side = section.side || section.dimensions?.side1 || 0;
      return 2 * (base + side);
    
    default:
      return 0;
  }
}
```

## 7. Миграция данных

### 7.1 Обратная совместимость

При загрузке старых проектов (без поля `shape`):
- `shape` устанавливается в `'rectangle'`
- Существующие поля `length` и `width` используются как есть

```typescript
function migrateSubSection(section: RoomSubSection): RoomSubSection {
  if (!section.shape) {
    return {
      ...section,
      shape: 'rectangle'
    };
  }
  return section;
}
```

## 8. Тестирование

### 8.1 Сценарии тестирования

1. **Прямоугольник**
   - Ввод длины и ширины
   - Проверка расчёта площади и периметра

2. **Трапеция**
   - Ввод всех параметров
   - Проверка формулы площади
   - Проверка формулы периметра

3. **Треугольник**
   - Ввод трёх сторон
   - Проверка формулы Герона
   - Граничный случай: невозможный треугольник (a + b < c)

4. **Параллелограмм**
   - Ввод основания, высоты, боковой стороны
   - Проверка расчётов

5. **Переключение форм**
   - Переключение между формами
   - Сохранение/очистка данных
   - Корректность расчётов после переключения

6. **Миграция**
   - Загрузка старого проекта
   - Автоматическое определение формы

### 8.2 Граничные случаи

- Нулевые значения
- Отрицательные значения (отклонить)
- Невозможный треугольник (предупреждение)
- Очень большие значения
