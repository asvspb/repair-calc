## 📋 Обзор

### Цель
Создать систему каталога типовых работ с автоматическим расчётом количества материалов, инструментов и расходников на основе метрик помещения.

### Проблемы, которые решает фича
1. **Ручной ввод повторяющихся данных** — пользователи создают одинаковые работы для разных комнат
2. **Ошибки в расчётах** — неправильный подсчёт количества материалов (краска, обои, плитка)
3. **Отсутствие подсказок** — нет рекомендаций по расходу материалов на основе площади
4. **Нет типовых решений** — каждый раз приходится заново подбирать материалы для работы

### Ключевые решения
| Решение | Описание | Приоритет |
|---------|----------|-----------|
| **Разные материалы на разных зонах** | Отдельные работы для каждой зоны | Высокий |
| **Слои краски** | Параметр `layers` в материале | Высокий |
| **Разные размеры** | Отдельные строки материалов | Высокий |
| **Подсказки расчёта** | Авто-расчёт с возможностью ручной корректировки | Высокий |
| **Каталог работ** | ~20 типовых работ без Lemana API (пока) | Высокий |
| **Общая смета** | Расширение текущей с группировкой по типам | Средний |
| **Цены через ИИ** | Gemini API для ориентировочных цен | Низкий |

### Не входит в текущий объём (Out of Scope)
- Интеграция с API Леруа Мерлен (отложено до получения доступа)
- Мобильное приложение
- Совместная работа над проектом
- Фотофиксация прогресса ремонта

## 📐 Модель данных

### Material (расширенный)

```typescript
interface Material {
  id: string;
  name: string;
  quantity: number;
  unit: string;
  pricePerUnit: number;

  // Параметры для расчёта (опционально)
  coveragePerUnit?: number;     // м² в упаковке/рулоне
  consumptionRate?: number;     // расход на м² (л/м², кг/м², шт/м²)
  layers?: number;              // количество слоёв (для краски)
  piecesPerUnit?: number;       // штук в упаковке (саморезы)
  wastePercent?: number;        // запас на подрезку %
  packageSize?: number;         // размер упаковки (л, кг) для подбора

  // Вычисляемое
  calculatedQty?: number;       // рекомендованное количество
  autoCalcEnabled?: boolean;    // использовать авторасчёт
}
```

**Правила валидации:**
- `quantity > 0` — всегда
- `coveragePerUnit` XOR `consumptionRate` — только одно для материала
- `layers` имеет смысл только с `consumptionRate`
- `wastePercent` в диапазоне 0–50

## 🔧 Формулы расчёта материалов

### Общие принципы

1. **Определение типа расчёта** — по `calculationType` работы
2. **Получение метрик помещения** — из `calculateRoomMetrics()`
3. **Применение формулы** — в зависимости от типа материала
4. **Учёт запаса** — `wastePercent` добавляется в конце
5. **Округление** — до разумных значений (2 знака после запятой)

### Формула 1: По площади покрытия

**Применение:** обои, ламинат, плитка, линолеум

```typescript
function calculateByCoverage(
  area: number,           // м² — площадь помещения
  coveragePerUnit: number, // м² в упаковке
  wastePercent: number    // % запаса
): number {
  const rawQty = area / coveragePerUnit;
  const withWaste = rawQty * (1 + wastePercent / 100);
  return Math.ceil(withWaste * 100) / 100; // округление до 2 знаков
}

// Пример: обои
// area = 24 м², coveragePerUnit = 5.3 м² (рулон), wastePercent = 10
// rawQty = 24 / 5.3 = 4.53
// withWaste = 4.53 * 1.10 = 4.98
// result = 5.00 рулонов
```

### Формула 2: По расходу на м²

**Применение:** краска, клей, затирка, штукатурка, грунтовка

```typescript
function calculateByConsumption(
  area: number,           // м² — площадь помещения
  consumptionRate: number, // ед/м² (л/м², кг/м², шт/м²)
  layers: number,         // количество слоёв (по умолчанию 1)
  wastePercent: number,   // % запаса
  packageSize?: number    // размер упаковки для подбора
): { total: number; packages: number } {
  const rawQty = area * consumptionRate * layers;
  const withWaste = rawQty * (1 + wastePercent / 100);
  const total = Math.ceil(withWaste * 100) / 100;
  
  const packages = packageSize 
    ? Math.ceil(total / packageSize)
    : 1;
  
  return { total, packages };
}

// Пример: краска
// area = 24 м², consumptionRate = 0.006 л/м², layers = 2, wastePercent = 5
// rawQty = 24 * 0.006 * 2 = 0.288 л
// withWaste = 0.288 * 1.05 = 0.302 л
// result = 0.30 л (или 1 банка 2.5 л)
```

### Формула 3: По периметру

**Применение:** плинтус, обрешётка, профили

```typescript
function calculateByPerimeter(
  perimeter: number,      // пог. м — периметр помещения
  multiplier: number,     // коэффициент (1.0 = 1:1)
  pieceLength?: number,   // длина 1 шт (для плинтуса)
  wastePercent: number    // % запаса
): number {
  const rawQty = perimeter * multiplier;
  const withWaste = rawQty * (1 + wastePercent / 100);
  
  // Если указана длина штуки — считаем количество штук
  const qty = pieceLength 
    ? Math.ceil(withWaste / pieceLength)
    : Math.ceil(withWaste * 100) / 100;
  
  return qty;
}

// Пример: плинтус
// perimeter = 18 м, multiplier = 1.0, pieceLength = 2.5 м, wastePercent = 5
// rawQty = 18 * 1.0 = 18 пог.м
// withWaste = 18 * 1.05 = 18.9 пог.м
// qty = ceil(18.9 / 2.5) = 8 штук
```

### Формула 4: Поштучно

**Применение:** уголки, заглушки, крепёж, точки электрики/сантехники

```typescript
function calculateByCount(
  count: number,          // количество
  multiplier: number,     // коэффициент
  wastePercent: number    // % запаса
): number {
  const rawQty = count * multiplier;
  const withWaste = rawQty * (1 + wastePercent / 100);
  return Math.ceil(withWaste); // округление до целых
}

// Пример: розетки
// count = 5, multiplier = 1.0, wastePercent = 10
// rawQty = 5 * 1.0 = 5
// withWaste = 5 * 1.10 = 5.5
// result = 6 штук (запас на случай повреждения)
```

### Формула 5: Объёмные материалы (стяжка, штукатурка)

**Применение:** материалы, где важна толщина слоя

```typescript
function calculateVolumetric(
  area: number,           // м² — площадь
  thickness: number,      // см — толщина слоя
  consumptionPerCm: number, // кг/м² на 1 см толщины
  wastePercent: number,   // % запаса
  packageSize: number     // кг в упаковке
): { totalKg: number; packages: number } {
  const rawKg = area * thickness * consumptionPerCm;
  const withWaste = rawKg * (1 + wastePercent / 100);
  const totalKg = Math.ceil(withWaste);
  const packages = Math.ceil(totalKg / packageSize);
  
  return { totalKg, packages };
}

// Пример: стяжка
// area = 20 м², thickness = 5 см, consumptionPerCm = 20 кг/м², wastePercent = 5, packageSize = 25 кг
// rawKg = 20 * 5 * 20 = 2000 кг
// withWaste = 2000 * 1.05 = 2100 кг
// packages = ceil(2100 / 25) = 84 мешка
```

### Кэширование результатов

```typescript
interface PriceCache {
  key: string; // hash(productName + city + category)
  result: PriceSearchResult;
  cachedAt: string;
  expiresAt: string; // +7 дней
}

// Кэш в localStorage
const PRICE_CACHE_KEY = 'repair-calc-price-cache';
const CACHE_TTL_DAYS = 7;

function getCachedPrice(key: string): PriceSearchResult | null {
  const cache = localStorage.getItem(PRICE_CACHE_KEY);
  if (!cache) return null;
  
  const entries: PriceCache[] = JSON.parse(cache);
  const entry = entries.find(e => e.key === key);
  
  if (!entry) return null;
  if (new Date(entry.expiresAt) < new Date()) {
    // Истёк — удалить
    const updated = entries.filter(e => e.key !== key);
    localStorage.setItem(PRICE_CACHE_KEY, JSON.stringify(updated));
    return null;
  }
  
  return entry.result;
}

function setCachedPrice(key: string, result: PriceSearchResult): void {
  const cache = localStorage.getItem(PRICE_CACHE_KEY);
  const entries: PriceCache[] = cache ? JSON.parse(cache) : [];
  
  // Удалить старую запись, если есть
  const filtered = entries.filter(e => e.key !== key);
  
  filtered.push({
    key,
    result,
    cachedAt: new Date().toISOString(),
    expiresAt: new Date(Date.now() + CACHE_TTL_DAYS * 24 * 60 * 60 * 1000).toISOString(),
  });
  
  localStorage.setItem(PRICE_CACHE_KEY, JSON.stringify(filtered));
}
```

### Ограничения и обработка ошибок

```typescript
interface PriceSearchError {
  type: 'network' | 'parse' | 'noResults' | 'rateLimit' | 'api';
  message: string;
  retryable: boolean;
}

async function searchPriceWithRetry(
  request: PriceSearchRequest,
  maxRetries = 2
): Promise<PriceSearchResult> {
  // 1. Проверка кэша
  const cached = getCachedPrice(buildCacheKey(request));
  if (cached) return cached;
  
  // 2. Попытки с retry
  let lastError: PriceSearchError | null = null;
  
  for (let attempt = 0; attempt <= maxRetries; attempt++) {
    try {
      const result = await searchPrice(request);
      setCachedPrice(buildCacheKey(request), result);
      return result;
    } catch (error) {
      lastError = mapErrorToPriceError(error);
      
      if (!lastError.retryable || attempt === maxRetries) {
        break;
      }
      
      // Exponential backoff
      await delay(1000 * Math.pow(2, attempt));
    }
  }
  
  throw lastError;
}
```

### Приложение A: Глоссарий

| Термин | Определение |
|--------|-------------|
| **Покрытие (coverage)** | Площадь, которую можно покрыть одной единицей материала |
| **Расход (consumption)** | Количество материала на 1 м² поверхности |
| **Слои (layers)** | Количество нанесений (для краски, лака) |
| **Запас (waste)** | Дополнительный процент на подрезку, брак, подгонку |
| **Пог.м (погонный метр)** | Единица измерения длины для плинтусов, профилей |

### Приложение C: Примеры расчётов

#### Пример 1: Расчёт обоев для комнаты 3×4 м, высота 2.7 м

**Дано:**
- Комната: 3×4 м, высота 2.7 м
- Окно: 1.5×1.5 м, Дверь: 0.9×2.0 м
- Обои: рулон 0.53×10 м = 5.3 м²
- Запас: 10% на подгонку рисунка

**Расчёт:**
1. Площадь стен: (3+4)×2×2.7 = 37.8 м²
2. Минус проёмы: 37.8 - (1.5×1.5 + 0.9×2.0) = 37.8 - 4.05 = 33.75 м²
3. Количество рулонов: 33.75 / 5.3 × 1.10 = 7.01 → **8 рулонов**

#### Пример 2: Расчёт краски для стен 50 м², 2 слоя

**Дано:**
- Площадь: 50 м²
- Расход: 0.006 л/м² на слой
- Слои: 2
- Запас: 5%
- Упаковка: 10 л

**Расчёт:**
1. Общий расход: 50 × 0.006 × 2 = 0.6 л
2. С запасом: 0.6 × 1.05 = 0.63 л
3. Количество банок: ceil(0.63 / 10) = **1 банка 10 л**

#### Пример 3: Расчёт плитки для пола 20 м², плитка 30×30 см

**Дано:**
- Площадь: 20 м²
- Плитка: 30×30 см = 0.09 м²
- Запас: 10% на подрезку
- В коробке: 9 шт = 0.81 м²

**Расчёт:**
1. Количество плиток: 20 / 0.09 × 1.10 = 244.44 → **245 шт**
2. Количество коробок: ceil(245 / 9) = **28 коробок**
3. Итого площадь: 28 × 0.81 = 22.68 м² (запас 13.4%)
