## 1. Текущее состояние

### ✅ Уже реализовано

1. **Типы данных**: `ProjectData` с `objects: ObjectData[]`, обратная совместимость через `rooms?: RoomData[]`
2. **ProjectContext**: `activeObjectId`, `activeObject`, CRUD-методы (`createObject`, `updateObject`, `deleteObject`, `copyObject`)
3. **API**: `syncPull()`, `updateProjectWithObjects()` — синхронизация объектов
4. **Компоненты**: `ObjectSelector`, `ObjectsList`, `ObjectCard`, `CreateObjectModal` — базовая реализация
5. **Утилиты**: `createNewObject`, `copyObjectInProject`, `deleteObjectFromProject` и др.

### ❌ Проблемы

1. **UI в `App.tsx` не интегрирует компоненты объектов**
2. **Путаница в терминологии**: "Объект" в UI = ProjectData, "Object" в коде = ObjectData
3. **BackupManager ("Данные") не интуитивен** — нужно переименовать в "Мои проекты"
4. **SummaryView не группирует по объектам**

## 5. Терминология

| Код (тип) | UI название | Описание |
|-----------|-------------|----------|
| `ProjectData` | Проект | Контейнер для объектов ремонта |
| `ObjectData` | Объект ремонта | Недвижимость (квартира, гараж, дом) |
| `RoomData` | Комната | Часть объекта ремонта |
