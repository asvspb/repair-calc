## Описание задачи

Необходимо выполнить следующие изменения в интерфейсе приложения:

1. **Переименовать кнопку "Данные" в "Настройки"** и изменить иконку на соответствующую названию
2. **Преобразовать модальное окно "Управление данными" в модальное окно с вкладками**:
   - Вкладка 1: "Управление данными" (существующий функционал)
   - Вкладка 2: "Пользователь" (перенести из правой боковой панели)
3. **Удалить секцию пользователя из правой боковой панели**

## Обоснование изменений

1. **Улучшение UX**: Логичное группирование настроек в одном месте
2. **Упрощение интерфейса**: Освобождение места в правой панели
3. **Консистентность**: Кнопка "Настройки" с соответствующей иконкой более понятна пользователю

## Риски и ограничения

- Минимальные риски, так как изменения касаются только UI
- Функциональность остается без изменений
