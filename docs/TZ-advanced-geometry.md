# Техническое задание: Режимы геометрии помещения

## 1. Общие сведения

**Проект:** Ремонт-Кальк (калькулятор стоимости ремонтных работ)  
**Версия:** 3.0  
**Дата:** 27.02.2026  
**Статус:** Реализовано

---

## 2. Цель и задачи

### 2.1 Цель
Расширить функциональность калькулятора для поддержки помещений сложной геометрической формы, что позволит точно рассчитывать площади и стоимость работ для реальных объектов.

### 2.2 Задачи
1. Добавить поддержку L-образных помещений, комнат с нишами и эркерами
2. Учесть препятствия (колонны, воздуховоды) в расчётах площади
3. Поддержать перепады высоты потолков
4. Обеспечить обратную совместимость с простым режимом

## 3. Функциональные требования

### 3.1 Режимы геометрии помещения

#### 3.1.1 Простой режим (по умолчанию)
- Прямоугольное помещение
- Ввод базовых размеров: длина, ширина, высота
- Расчёт по стандартным формулам
- Окна и двери указываются для всего помещения

#### 3.1.2 Расширенный режим (новое в версии 3.0)
- Включается переключателем в секции "Габариты помещения" (кнопки "Простой" / "Расширенный" / "Профессиональный")
- Предназначен для L-образных и Г-образных помещений
- Помещение разбивается на прямоугольные секции
- Каждая секция имеет свои:
  - Название
  - Длину и ширину
  - Окна и двери
- Расчёты:
  - Площадь пола = Σ(площади всех секций)
  - Периметр = Σ(периметры всех секций)
  - Проёмы суммируются из всех секций
- Идеально подходит для:
  - L-образных комнат
  - Квартир-студий
  - Помещений с выступами

**Структура данных:**
```typescript
type RoomSubSection = {
  id: string;           // Уникальный идентификатор
  name: string;         // Название (например: "Основная часть", "Эркер")
  length: number;       // Длина секции (м)
  width: number;        // Ширина секции (м)
  windows: Opening[];   // Окна этой секции
  doors: Opening[];     // Двери этой секции
};
```

#### 3.1.3 Профессиональный режим
- Включается переключателем в секции "Габариты помещения"
- Добавляет три дополнительных блока:
  - Сегменты помещения
  - Препятствия
  - Перепады высоты стен
- Предназначен для помещений сложной формы с нестандартными элементами

### 3.2 Сегменты помещения

**Назначение:** Учёт дополнительных площадей для помещений сложной формы

**Структура данных:**
```typescript
type RoomSegment = {
  id: string;           // Уникальный идентификатор
  name: string;         // Название (например: "Ниша", "Эркер")
  length: number;       // Длина сегмента (м)
  width: number;        // Ширина сегмента (м)
  operation: 'add' | 'subtract';  // Добавить или вычесть площадь
};
```

**Функционал:**
- Добавление неограниченного количества сегментов
- Редактирование названия, размеров
- Выбор операции: добавить или вычесть площадь
- Отображение итоговой площади сегментов
- Удаление сегментов

**Влияние на расчёты:**
- Площадь пола: ± длина × ширина
- Периметр: ± 2 × (длина + ширина) — приблизительно
- Площадь стен: пересчитывается через периметр

### 3.3 Препятствия

**Назначение:** Учёт элементов, занимающих площадь в помещении

**Структура данных:**
```typescript
type Obstacle = {
  id: string;           // Уникальный идентификатор
  name: string;         // Название (например: "Колонна 1")
  type: 'column' | 'duct' | 'niche' | 'other';  // Тип препятствия
  area: number;         // Площадь (м²)
  perimeter: number;    // Периметр (м) — влияет на плинтуса
  operation: 'add' | 'subtract';  // Добавить или вычесть
};
```

**Типы препятствий:**
- `column` — Колонна
- `duct` — Воздуховод/короб
- `niche` — Ниша в стене
- `other` — Другое

**Функционал:**
- Добавление неограниченного количества препятствий
- Выбор типа из справочника
- Ввод площади и периметра вручную
- Выбор операции

**Влияние на расчёты:**
- Площадь пола: ± area
- Длина плинтусов: ± perimeter

### 3.4 Перепады высоты стен

**Назначение:** Учёт участков стен с отличающейся высотой

**Структура данных:**
```typescript
type WallSection = {
  id: string;           // Уникальный идентификатор
  name: string;         // Название (например: "Стена с перепадом")
  length: number;       // Длина участка (м)
  height: number;       // Высота участка (м)
};
```

**Функционал:**
- Добавление неограниченного количества участков
- Ввод длины и высоты участка
- Автоматический перерасчёт площади стен

**Влияние на расчёты:**
- Площадь стен корректируется: 
  - Вычитается: длина × базовая высота
  - Добавляется: длина × высота участка

## 4. Алгоритмы расчёта

### 4.1 Основная функция calculateRoomMetrics

```typescript
function calculateRoomMetrics(room: RoomData) {
  let floorArea = 0;
  let perimeter = 0;
  let totalWindowsArea = 0;
  let totalDoorsArea = 0;
  let totalDoorsWidth = 0;
  
  // Расширенный режим: суммируем все секции
  if (room.geometryMode === 'extended' && room.subSections?.length > 0) {
    for (const section of room.subSections) {
      floorArea += section.length * section.width;
      perimeter += (section.length + section.width) * 2;
      
      // Суммируем проёмы секции
      for (const window of section.windows || []) {
        totalWindowsArea += window.width * window.height;
      }
      for (const door of section.doors || []) {
        totalDoorsArea += door.width * door.height;
        totalDoorsWidth += door.width;
      }
    }
  } else {
    // Простой и профессиональный режим: базовые расчёты
    floorArea = room.length * room.width;
    perimeter = (room.length + room.width) * 2;
    
    totalWindowsArea = room.windows.reduce((sum, w) => sum + w.width * w.height, 0);
    totalDoorsArea = room.doors.reduce((sum, d) => sum + d.width * d.height, 0);
    totalDoorsWidth = room.doors.reduce((sum, d) => sum + d.width, 0);
  }
  
  let grossWallArea = perimeter * room.height;
  
  // В профессиональном режиме
  if (room.geometryMode === 'advanced') {
    // Сегменты
    floorArea += segmentsDelta;
    perimeter += segmentsPerimeterDelta;
    
    // Препятствия
    floorArea += obstaclesAreaDelta;
    perimeter += obstaclesPerimeterDelta;
    
    // Пересчёт площади стен
    grossWallArea = perimeter * room.height;
    grossWallArea += wallSectionsDelta;
  }
  
  // Финальные расчёты
  const netWallArea = grossWallArea - totalWindowsArea - totalDoorsArea;
  const skirtingLength = perimeter - totalDoorsWidth;
  
  return { floorArea, perimeter, netWallArea, skirtingLength };
}
```

### 4.2 Формулы

**Простой режим:**
| Параметр | Формула |
|----------|---------|
| Площадь пола | длина × ширина |
| Периметр | 2 × (длина + ширина) |
| Площадь стен | периметр × высота - площадь окон - площадь дверей |
| Длина плинтусов | периметр - Σ(ширина дверей) |

**Расширенный режим:**
| Параметр | Формула |
|----------|---------|
| Площадь пола | Σ(длина секции × ширина секции) |
| Периметр | Σ(2 × (длина секции + ширина секции)) |
| Площадь стен | периметр × высота - Σ(площадь окон всех секций) - Σ(площадь дверей всех секций) |
| Длина плинтусов | периметр - Σ(ширина дверей всех секций) |

**Профессиональный режим:**
| Параметр | Формула |
|----------|---------|
| Площадь пола | baseFloorArea + Σ(segment.area × sign) + Σ(obstacle.area × sign) |
| Периметр | basePerimeter + Σ(segment.perimeter × sign) + Σ(obstacle.perimeter × sign) |
| Площадь стен | perimeter × height + Σ(wallSection.delta) - windowsArea - doorsArea |
| Длина плинтусов | perimeter - Σ(door.width) |

## 5. Пользовательский интерфейс

### 5.1 Расположение элементов

**Общие элементы:**
1. **Переключатель режима** — справа от заголовка "Габариты помещения" (три кнопки)

**Расширенный режим:**
1. **Поля длины и ширины** —隐藏ы, не используются для расчётов
2. **Секции помещения** — отдельная секция ниже габаритов
   - Каждая секция: название, длина, ширина
   - Внутри секции: блоки окон и дверей
3. **Блоки окон/дверей** — скрыты на уровне помещения, редактируются внутри секций

**Профессиональный режим:**
1. **Сегменты** — отдельная секция ниже габаритов
2. **Препятствия** — отдельная секция после сегментов
3. **Перепады высоты** — отдельная секция после препятствий

### 5.2 Визуальные элементы

- Переключатель: кнопки "Простой" / "Расширенный" / "Профессиональный" в стиле табов
- Предупреждение при простом режиме: жёлтый блок с иконкой AlertCircle
- Предупреждение при переключении из расширенного режима: информация о потере данных секций
- Карточки элементов: серый фон, скруглённые углы
- Итоговые значения: внизу каждой секции
- Секции в расширенном режиме: нумерованный список с возможностью сворачивания

### 5.3 Адаптивность

- На мобильных устройствах: поля ввода переносятся на новую строку
- Кнопки действий: компактные, с иконками

## 6. Хранение данных

### 6.1 Обновлённая структура RoomData

```typescript
type GeometryMode = 'simple' | 'extended' | 'advanced';

type RoomData = {
  id: string;
  name: string;
  geometryMode: GeometryMode;
  length: number;
  width: number;
  height: number;
  subSections: RoomSubSection[];        // NEW в v3.0 — секции для extended режима
  segments: RoomSegment[];              // для advanced режима
  obstacles: Obstacle[];                // для advanced режима
  wallSections: WallSection[];          // для advanced режима
  windows: Opening[];                   // для simple и advanced режима
  doors: Opening[];                     // для simple и advanced режима
  works: WorkData[];
};
```

### 6.2 Миграция данных

При загрузке старых проектов:
- `geometryMode` по умолчанию = 'simple'
- Пустые массивы для subSections, segments, obstacles, wallSections
- Обратная совместимость: старые проекты работают без изменений

## 7. Тестирование

### 7.1 Сценарии тестирования

1. **Простой режим**
   - Создание нового помещения
   - Ввод базовых размеров
   - Проверка расчётов

2. **Расширенный режим (новое в v3.0)**
   - Переключение в расширенный режим
   - Добавление секций помещения
   - Добавление окон и дверей в секции
   - Проверка суммарных расчётов
   - Переключение обратно — предупреждение о потере данных

3. **Переключение режимов**
   - Переключение из простого в профессиональный
   - Ввод данных в профессиональном режиме
   - Переключение обратно — проверка предупреждения

4. **Сегменты**
   - Добавление сегмента с операцией "+"
   - Добавление сегмента с операцией "-"
   - Проверка итоговой площади
   - Удаление сегмента

5. **Препятствия**
   - Добавление препятствия разных типов
   - Проверка влияния на площадь и периметр
   - Удаление препятствия

6. **Перепады высоты**
   - Добавление участка с большей высотой
   - Добавление участка с меньшей высотой
   - Проверка корректности расчёта площади стен

### 7.2 Граничные случаи

- Все значения = 0
- Отрицательные площади (через вычитание)
- Очень большие значения
- Пустые названия

## 8. Ограничения и известные проблемы

1. **Периметр сегментов** — рассчитывается приблизительно (как прямоугольник)
2. **Нет визуализации** — отсутствует графическое отображение геометрии
3. **Нет валидации** — можно ввести некорректные значения (например, отрицательные)
